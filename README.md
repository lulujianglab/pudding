# pudding

关键功能

- [x] 自动 resize 
- 复制图片
- [x] 默认欢迎文章(各种格式)
- [x] Ctrl + S 保存

竞争对手分析

- gridea
- Typora
- webm
- maxiang.io
- evernote
- picgo 图床
- eagle 图片管理

我们的优势:

- 代码高亮, 基于 vscode 编辑器
- 全文检索
- 自动标签
- 可配置打通 oss, cdn 等
- 支持生成长微博
- 生成 pdf
- 支持发布 github
- 可发布到语雀
- 可导入 github issues

准备支持的 markdown 特性

- 数学公式
- 图片尺寸
- 图表, 流程图
- GFM 任务清单
- GFM 表格
- markdown-it-emoji
- 文章目录
